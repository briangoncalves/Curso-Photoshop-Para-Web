## Conteúdo

Os principais pontos tratados serão:

- Conhecendo o Photoshop
- Ferramentas Principais
- Como usar as ferramentas
- Construindo objetos com Photoshop para web
- Desenvolvimento de Wireframe
- Montagem do Design
- Criação de Layout
- Passar PSD/HTML
- Uso de Framework
- Uso de Mockup
